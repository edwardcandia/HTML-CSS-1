Formatações adicionais em HTML

Marcação de Texto

o termo em destaque se usa utilizando a tag <mark>.

Texto grande e texto pequeno

criamos o texto grande utilizando-se a tag <big>.

criamos o texto pequeno utilizando-se a tag <small>.

Texto excluído

criamos o texto excluído utilizando-se a tag <del>.

Texto inserido

criamos o texto inserido utilizando-se a tag <ins>.

Texto sobrescrito

Exemplo: x15+20=y.

Basta usarmos a tag <sup>.

Texto sobrescrito

Exemplo: H20.

Basta usarmos a tag <sub>.
